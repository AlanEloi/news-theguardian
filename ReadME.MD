# The Guardian
Programa que consume a API do jornal The Guardian, mostrando o título e o link das notícas do dia.

  API utilizada: [The Guardian](https://open-plataform.theguardian.com)

### Biblioteca para funcionamento do script
```sh
pip install request
```
```sh
pip install json
````
````sh
pip install pandas
````
